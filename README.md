# CV y Portafolio Web

## Descripción

Proyecto desarrollado como parte de la actividad de GitHub Pages.  
El objetivo fue construir un CV web utilizando HTML y CSS, incorporando secciones principales como navegación, educación, experiencia, portafolio y footer.

## Sitio desplegado

GitHub Pages:  
https://danielamontecinoslarrain.github.io/

## Requisitos de la actividad

### 1. Construcción del CV con HTML y CSS

El sitio incluye:

- Barra de navegación.
- Sección de presentación / perfil profesional.
- Sección de educación.
- Sección de experiencia.
- Link o acceso al portafolio.
- Footer.

### 2. Construcción del portafolio

El proyecto incluye una sección de portafolio con imágenes de trabajos anteriores, permitiendo mostrar proyectos visuales y experiencia previa.

### 3. Despliegue en GitHub Pages

El CV fue desplegado en GitHub Pages, configurando el archivo principal como página de inicio del sitio.
